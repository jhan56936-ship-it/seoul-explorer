# 🌆 Seoul Explorer (서울 익스플로러)

**Seoul Explorer**는 웹 브라우저 위에서 실행되는 **2D 오픈월드 어드벤처 게임**입니다. 플레이어는 광활하게 구현된 서울 맵을 자유롭게 돌아다니며 실제 서울의 유명한 랜드마크와 명소들을 탐험할 수 있습니다.

---

## 🛠️ 기술 스택 (Tech Stack)
* **프론트엔드 프레임워크**: React, Vite, TypeScript
* **그래픽 및 렌더링**: HTML5 `<canvas>` API, 순수 JavaScript 수학 연산 (충돌 및 절차적 맵 생성)
* **스타일링**: CSS (자체 구현된 사이버펑크 느낌의 Glassmorphism UI)
* **아이콘**: Lucide-react

---

## 🚀 주요 기능 및 특징 목록

### 1. 거대한 오픈월드 맵과 미니맵 시스템
* 맵 크기: `120,000px × 120,000px`의 매우 방대한 심리스 맵 구조.
* 한강 및 지형이 전체 맵에 걸쳐 유려하게 그려져 있습니다.
* 우측 상단의 반투명 미니맵을 통해 전체 지형 속 자신이 어디에 있는지 실시간으로 파악할 수 있습니다.
* 플레이어의 움직임에 맞춰 카메라가 부드럽게 중앙을 따라다닙니다(Camera Tracking).

### 2. 52곳 이상의 디테일한 서울 랜드마크 (Landmarks)
경복궁, 롯데월드타워, 코엑스, 명동거리, N서울타워 등 서울을 대표하는 핵심 명소 52곳 이상이 모두 맵 구역에 알맞게 배치되어 있습니다. 단순한 배경 요소가 아니라 물리적 판정을 지닌 **실제 탐험 가능한 구역**입니다.
* 안전한 스폰: 초기 스폰 위치는 `x: 56000, y: 48000` (광화문/시청 부근 안전지대) 입니다.
* 간격 유지: 모든 명소는 건물 겹침 현상을 방지하기 위해 최소 500px 이상의 간격을 두고 띄엄띄엄 배치되었습니다.

### 3. 출입 게이트 및 내부 절차적 생성 로직 (Procedural Interiors)
건물이나 공원은 외부에서 보이지 않는 두꺼운 **물리적인 타일 기반 벽**으로 둘러싸여 있으며, 모든 명소의 밑면에는 `ENTRANCE`라고 적힌 출입구가 열려있습니다. 출입구로 진입하면 다음과 같은 내부 배치가 절차적으로(랜덤에 기반하지만 고정되게) 생성되어 있습니다.
* **자연/공원 (숲, 강, 산)**: 무작위 산림 구역(Tree)들이 무수하게 깔려 있습니다.
* **고궁 및 한옥**: 전통적인 궁궐 모양의 큼직한 전각(Pavilions)들이 마당을 끼고 일정하게 마주보고 배치됩니다.
* **박물관 및 쇼핑몰**: 부스(Booths)나 방(Rooms)과 같은 박스형 방들이 복도를 이루며 미로처럼 배치되어 있습니다.
* **충돌 판정 (Collision)**: 건물 외벽은 물론, 내부에 배치된 구조물들 또한 물리적 충돌 판정을 가지므로 플레이어가 뚫고 지나갈 수 없습니다.

### 4. 세련된 Glassmorphism UI
* **사이버펑크/모던 룩**: 우주선 대시보드를 연상케 하는 반투명한 패널들이 현재 플레이어의 X, Y 좌표 실시간 전송, 근처 랜드마크 발견 시스템 알림을 부드럽게 띄워줍니다.

### 5. 캐릭터 조작 (Movement)
* **방향 이동**: `W`, `A`, `S`, `D` 또는 `방향키`를 사용합니다.
* **달리기(Sprint)**: 맵이 매우 거대하기 때문에 `Shift` 키를 누른 상태에서 이동하면 플레이어가 원래 속도보다 3배 이상 빠르게 질주하여 거대한 건물 내부를 시원하게 누빌 수 있습니다.

## 🎯 앞으로의 확장 가능성 (Roadmap)
이 프로젝트는 웹 기반 2D 엔진의 기초적인 뼈대(Engine Base)를 아주 탄탄하게 완성한 상태이기 때문에, 다음과 같이 기능을 덧붙일 수 있습니다.
* 특정 명소(예: 식당 건물) 내부에 들어가면 관련된 **퀴즈나 미니게임** 팝업 열기.
* 다른 캐릭터(NPC)를 맵 곳곳에 배치하고 만나면 텍스트로 대화하기.
* 숨겨진 상자나 코인을 배치하는 수집 콘텐츠 추가.
